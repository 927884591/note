# css简介

由于css历史的原因，css有些标签中内联样式可以写一些属性，例如color，但有一些标签写上去上没有效果的，后来为了统一标准，诞生了css

# 选择器

  1. 标签选择器

# 字体单位

  1. px。 //像素
  2. em。 相对于当前对象字体的尺寸

# id选择器和类选择器

他们有次数的区别

类可以重复的使用

id就是只能用一次

# 伪类选择器

指的是标签的某种状态后指定的样式

# 块级显示标签

块级显示的特点就是独占一行，且可以设置宽度高度，对齐等属性

常见的块级显示标签

  1. div
  2. h1-h6
  3. p
  4. ul
  5. ol
  6. li

# 行内元素

行内元素就是靠内容本身撑大，一般不可以设置高度，宽度，对齐等属性，行内元素只能放文字和行内元素不能放块级元素，对行内元素使用margin只能是左右不能上下

常见的行内元素有

  1. a。
  2. strong
  3. b
  4. em
  5. i
  6. del
  7. s
  8. ins
  9. span

# 行内块元素

常见的行内块元素有

  1. img
  2. input
  3. td

# background属性

//以下是简写

  1. bgcolor
  2. bgrepeat
  3. bgposition。
  4. bgattachment。 //默认值是scroll。 还有个fixed值。 fixed是背景固定，滚动背景图不动
  5. bg-size //可以写px单位。 建议只写一个。因为是等比例缩放。 还有一个值是cover cover始终是填满背景图。 Contain 。 保证图片显示是完整的
  6. bg //可以加多张背景图片，每张图片以逗号分隔开

# css的层叠性

如果样式起冲突会以最后的样式生效。

# css的继承性

子元素标签会继承父元素标签的属性，继承的权重为0

只有部分属性才会继承，例如text- ，font-，line- 以及color这些开头的都会继承

# css的优先级

  1. 标签选择器 1
  2. 类选择器 10
  3. id选择器 100
  4. 行内选择器 1000
  5. !important 无穷大

# float

  1. 如果给块级作用域添加float，这个块级元素就会有行内块元素的特点，内容撑开，而不是独占一行
  2. 如果给行内元素添加也会有行内块元素的特点

# 清除浮动的方法

清除浮动是指清除浮动的影响，而不是让它回到普通的文档流

浮动也会导致父级元素高度塌陷的问题，子元素使用浮动，父级元素如果没有设置高度的话，就会导致父级元素的高度上0

  1. 在浮动的最后添加一个一个元素，给这个元素添加一个clear:both;的属性
  2. 给父级元素添加overflow：hiden；属性
  3. ::after伪元素添加clrear:both

# 定位

给元素添加了绝对定位或固定定位后，元素的显示模式就会切换成行内块模式

  1. absolute。 //绝对定位
  2. fixed。 固定定位
  3. relative。 相对定位
